#Project Overview: A React-based front-end application for browsing, sorting, and interacting with news articles. Users can view topics, read full articles, post comments, vote on articles, and navigate through a responsive, user-friendly interface. This project connects to the NC News REST API built in the back end (link included below).

#Project Link: https://nc-news-app-react.netlify.app/

#Backend API Link: https://seeding-nc-news-097y.onrender.com/

#Minimum version to run Nodejs: v12

#Clone this project locally by using the command below:

git clone https://github.com/fariahasan36/nc-news.git

#Install dependencies:

To install necessary dependencies listed in package.json use the command npm install in the project root directory
To install dev dependencies listed in package.json use the command npm install --save-dev in the project root directory

#Run the project: Run the project by using the command below

npm run dev
